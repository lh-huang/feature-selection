# feature-selection

This repository contains the code and datasets for our feature selection studies.

## 1. Unsupervised Feature Selection

This repository currently provides the code for the following paper:

**FWHSR: An unsupervised feature selection framework via feature-weighted hypergraph and clustering similarity self-representation**

The paper has been accepted/published and is available at:

[https://doi.org/10.1016/j.neucom.2026.133507](https://doi.org/10.1016/j.neucom.2026.133507)

The corresponding code and datasets are provided in this repository.

## 2. Multi-Label Feature Selection

We are also preparing a new paper on multi-label feature selection:

**TSSFS: Triple-Layer Self-Representation Learning with Shared Structural Information for Multi-Label Feature Selection**

The code, datasets, and detailed usage instructions for TSSFS will be released in this repository after the paper is accepted/published.

## Updates

More details, code, and usage instructions will be continuously updated in this repository.
